    Babel is a JavaScript compiler. babel-plugin-class-bound-components is a plugin which detects code using the class-bound-components package and applies some convenience transformations to it.

    E.g., it infers the display name of components created with the classBound function and inserts it into the correct position in the signatures in the compiled code. If you cannot use ES6 Proxy in your client-side codebase, it transforms calls to the classBound[JSX.IntrinsicElement]() helper signature to one that can be used without erroring on browsers without Proxies, like IE11.

    In WordPress, the so called Shortcodes are powerful building blocks to include rich content in posts and pages. Nevertheless, through the XML like syntax they are not so easy to assemble for non-technical users.
